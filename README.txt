== Description ==

<h3>Admin Guide Dashboard Widget</h3>This is a simple plugin for adding  posts in the admin dashboard from category set/define by you.
<p>There are several instance where you want to store and share notes, instructions, site's policies for administrating your site. It may be for yourself to keep remember some admin stuff or for other administrators. In some case site is administrate by more than single <strong>administrator</strong> in that case it will be useful to create <strong>guide</strong> to <strong>administrate</strong> site.</p>
<p>It's very easy to set up. Create category with name admin or what you feel suitable. From "<strong>Admin Guide Dashboard Widget</strong>" configuration form enter category id to set it. And post all your notes, instructions, site's policies into that category. You Done it! You will get handy admin <strong>guide</strong> into WordPress dashboard. Don't forget to keep all those posts as "Private" while publishing posts.</p>

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Admin Guide Dashboard Widget'
3. Click 'Install Now'
4. Activate the plugin on the Plugin dashboard

= Uploading in WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Navigate to the 'Upload' area
3. Select `admin-guide-dashboard-widget.zip` from your computer
4. Click 'Install Now'
5. Activate the plugin in the Plugin dashboard

= Using FTP =

1. Download `admin-guide-dashboard-widget.zip`
2. Extract the `admin-guide-dashboard-widget` directory to your computer
3. Upload the `admin-guide-dashboard-widget` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard
